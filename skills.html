user research

The most important aspect of user research for me is that it allows for a departure from merely anticipating the needs of the user in favor of understanding the user. This is done by setting aside preconceived notions.

user empathy

Instead of just imagining what it is like to be a user, it should be the goal of a UX designer to give voice to the feelings, needs, and motivations driving the user through an experience.
